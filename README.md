## Instalation

1. Clone the repository:
```bash
git clone https://github.com/brubrux999/Blackjack.git
```

2. Go to the project directory:
```bash
cd Blackjack
```

3. Run the game:
```bash
python Blackjack.py
```

## How to play

- Select an option from the menu.
- Choose whether to Hit (h) or Stand (s).
- Try to reach 21 points or get as close as possible without going over.
- The computer acts as the dealer following soft 17 rules.
- The one closest to 21 points wins.
- Choose Yes (y) if you want to play again or No (n) if you don't.
